Dfa2Regex

Description

Converts a deterministic finite automaton (DFA) into a regular expression (Regex), using the DFA2Regex function. This program is based on the Transitive Closure Method, which is an recursive algorithm to find regular expressions associated to a deterministic finite automaton.
